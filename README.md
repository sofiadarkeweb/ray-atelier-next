## Ray Atelier

Site for spatial design agency Ray Atelier Stockholm. A minimalistic site with the images in focus.

### Tech

Next.js 16, React 19, and headless CMS Contentful. NPM package Masonry for react. Deployed on Vercel.

Use **Node 24** locally (see `.nvmrc`, e.g. `nvm use`). In Vercel project settings, set the Node.js version to **24.x** to match.

`legacy-peer-deps` is set in [`.npmrc`](.npmrc) so `react-headroom` (no React 19 peer range yet) can install alongside React 19.
